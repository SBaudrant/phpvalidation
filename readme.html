'.template' templating engine


How it works


Use the following tags to format your text, multiple arguments being separated by ' , '



Titles


To create titles: '#1', '#2', ... then the title.



Blocks


To create a block: 'bl ' then the background color, then the (optional) text color.



Lists


To creat lists, start each item of the list with '-- ' on a new line.


example:

not in list

  • item 1
  • item 2
  • item ...

not in list



Links


To create a link, use 'l: ' folowed by the address, then the text.

The text is optional.


example:

This one is with a text

This one without:

https://google.com


Images


To place an image, use 'i: ' folowed by the address, then the (optional) alt text.

Centered text


To center text, use 'ct'.


example:

This text is centered.

To create a separating line, use '___'.
